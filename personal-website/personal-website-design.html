Project plan

Title

Natural Moments

Goal

This website exists to showcase my photography. Share my photos with people who enjoy nature.

Target Audience

Friends or family who want to see my photography. People who enjoy photography and/or nature.

Main Features

Photographs, about me page, contact forum, and descriptions of photos.

Personality

The mood of the site will be peaceful, vibrant, cheerful, and nature.

I think it will create an inviting mood towards the viewers, give them an uplifting mood.

Limiting Factors

I am not sure if all my photos will be in the highest resolution or largest size.

Styles guide

Background

Background is light olive green

{background-color: #cfdacc;}

Default font

Text is coal black, 14 pixels

p { font-size: 14px; color: #483e49;}

Header

The header section is 235px tall, positioned along the top of each page. The background is Burgundy.

header {
height: 235px;
background-color: #8C5A63;
background-image: url("/images/whale.png");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 120px tall, positioned just below the header with a background color of Light Oliver Green.

nav { height: 120px; background-color: #CFDACC; }

Images

Images will be over light pink background with a grey border around the container

figure { background-color: #6699CC; border: 2px solid #333333; }

img { padding: 1em; }

Links

Links will be Teal. On hover, they will have a red background and a light pink border.

a { color: #76A1A6 }

a:hover { background-color: #fflala; border 2px dotted ##F2C9D1; }
