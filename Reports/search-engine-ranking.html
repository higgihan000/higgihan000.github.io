A search engine works by using a specific algorithm which can be a formula or set of rules. These algorithms help filter out the spam pages and bring you the real deal. A major thing that every algorithm checks for while filtering out websites is relevancy. Why do different search engines give different ranking to websites for the same search phrase?

Different search engines give you different results for the same exact phrases being searched because each one has different factors that lead to their top ranking website results. Most common factor is the search engines are looking for high quality content. Does the webpage stay on topic and not stray? Is there lots of information about the keywords used. Is it easy to read? Is there graphics with the webpage?

What is a meta tag? What is the purpoes for Keyword and Description meta tags?

Meta tags are information about the pages context that you can only see through the pages code. It tells the search engines what this webpage is about and some context. It is good to have keyword and description meta tags so it is more likely for a search engine to correctly find your webpage. You want to include meta tags in your webpage because it helps your page become more easily searchable. Yes it improves your websites ranking.
