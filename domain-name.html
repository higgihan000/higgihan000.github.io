Domain Name Report

What is an Internet Domain Name?

A name that people use to get to your website. They are linked to IP addresses, but it is easier to remember names more than numbers so they use domain names.

Why are domain names so important to the web?

Without them their would be no proper organization to the internet and would be much less effective.

Why do domain names have to be unique?

People remeber unique names more and it is more likely to be available. Easier to find your website.

What does it mean to "register" a domain nam?

It means checking to see if that domain name is available and then purchasing it for a certain amount of time.

If you wanted a domain name that is already taken, how could you get it?

You can buy it in the domain aftermarket.
